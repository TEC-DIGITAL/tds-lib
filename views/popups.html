HTML

Use the following html structure for create a tds-lib-pop-up, and don't forget use togglePopUp( id_pop-up ) function, sending by parameter the id of the element that you want to hide:

<div id ="id_ejemplo" title="Helvetica Bold 16px (12pt)">
    <p class="tds-lib_text_01">Helvetica 15px (11 pt)</p>
    <p class="tds-lib_text_01">Leading 21px (15 pt)</p>
</div>

Javascript

Use the following code to create an instance of tds-lib-pop-up

new  tds_lib.PopUp( document.getElementById("id_ejemplo"),{ accept: varAccept, cancel: varCancel } );

The arguments for tds_lib.PopUp constructor are, the object and a configuration object with the following sentence: { accept: functionAccept, cancel: functionCancel }

functionCancel and functionAccept are closures, for example:

var varAccept = function () {
    alert("soy el botón aceptar y esto es una pruba.");
    return true;
};

var varCancel = function () {
    alert("soy el botón cancelar y esto es una prueba.");
    return false;
};

If you want that the action of the botton close the pop up, the closure must return true, if the closure return false, the pop up will continue opened.
