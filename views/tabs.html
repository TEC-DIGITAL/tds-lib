The tds-lib-tabs allows you to group data inside a tab.

How to use it?

First you have to add the webcomponent library

			
	<script src="/resources/tds-lib/lib/webcomponentsjs/webcomponents-lite.min.js"></script>

Then include the tab web component in your project. You can add this line in any part of your HTML or ADP file.

			
	<link rel="import" href="/resources/tds-lib/web_components/tds-lib-tab/tds-lib-tab.html">

That's all! Now you can use the tds-lib-tab element.

The .hide-till-loaded class will be deleted when the element is ready. So you can add any css property you need, like a placeholder or hide the element.

Add a tab using JavaScript

To add a tab using JavScript you can use the traditional method

			
	// Create tab with JavaScript
	var node = document.createElement("tds-lib-tab");
	node.setAttribute("tabname","Added with JS");
	document.body.appendChild(node);
	// Show only if the element is ready
	node.className = "hide-till-loaded";
	// Add content to tab with JavaScript
	var textnode = document.createTextNode("Tab content");
	node.appendChild(textnode);

Events
Name Params Description Example
set(objectOrText) objectOrText: Can be an HTML string or an object. Changes the content of the tab with the objectOrText content. 

document.querySelector("#myTab").setContent("
Hello World!
");
document.querySelector("#mitab").setContent(document.querySelector("#content"));
appendChild(objectOrText) objectOrText: Can be an HTML string or an object. Adds an element at the end of the tab content 

document.querySelector("#myTab").appendChild("
Hello World!
");
document.querySelector("#mitab").appendChild(document.querySelector("#content"));
toggleState() Enable / Disable tab 

document.querySelector("#myTab").toggleState();
openTab() Open a specific tab 

document.querySelector("#myTab").openTab();
closeTab() Close a specific tab 

document.querySelector("#myTab").closeTab();
change() Fires when a tab changes its state 

document.querySelector("#myTab").addEventListener("change", function(data){
	// The change info will be in data.details
	// Ex
	console.log(data.detail.open);
});
