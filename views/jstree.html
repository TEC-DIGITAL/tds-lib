jsTree

jsTree is jquery plugin, that provides interactive trees. It is absolutely free, open source and distributed under the MIT license. jsTree is easily extendable, themable and configurable, it supports HTML & JSON data sources and AJAX loading.

Example:

Including jsTree

Include the following code into the head section of your site's HTML. If you want to use the jsTree elements.

<script src="/resources/tds-lib/lib/jstree/jstree.min.js"></script>

Code example:

<div id="jstree">
    <ul>
        <li>Root node 1
            <ul>
                <li id="child_node_1">Child node 1</li>
                <li>Child node 2</li>
            </ul>
        </li>
        <li>Root node 2</li>
    </ul>
</div>

When DOM is ready, create an instance of jstree with the following javascript code:

$('#divMenu').jstree();
