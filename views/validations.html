Number Input

Use 'tds-lib_form' class for Number Input.

HTML

<div class="form-group">
<code style="margin-left:px;"> <label for="numberInput" class="tds-lib_form">Input Label</label></code><br>
<input id="numberInput" type="text" class="tds-lib_form" placeholder="Enter the text">
</div>

JavaScript

Get the input element that you want to use as number input

 var mynumberInput = new tds_lib.NumberInput(document.getElementById("numberInput"));

Methods

mynumberInput.setMaxValue(1000);
mynumberInput.setMinValue(-50);
mynumberInput.setInputValue("-40,2");
