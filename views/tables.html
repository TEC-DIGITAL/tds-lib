Datatable

Use 'display' class for Datatable style.

HTML

<table id="dataTableExample" class="display" cellspacing="0" width="100%">
        <thead>
            <tr>
                <th>Name</th>
                <th>Position</th>
                <th>Office</th>
                <th>Age</th>
                <th>Start date</th>
                <th>Salary</th>
            </tr>
        </thead>
        <tfoot>
            <tr>
                <th>Name</th>
                <th>Position</th>
                <th>Office</th>
                <th>Age</th>
                <th>Start date</th>
                <th>Salary</th>
            </tr>
        </tfoot>
        <tbody>
            <tr>
                <td>Tiger Nixon</td>
                <td>System Architect</td>
                <td>Edinburgh</td>
                <td>61</td>
                <td>2011/04/25</td>
                <td>$320,800</td>
            </tr>
        </tbody>
    </table>

JavaScript

Get the table element that you want to transform to dataTable

 var dataTable = $("#dataTableExample").datatable();

Bordered table

To use the standard for tables, add class="table table-striped table-bordered" in the <table> html tag.
For example:

<table class="table table-striped table-bordered">
        <thead>
            <tr>
                <th>Firstname</th>
                <th>Lastname</th>
                <th>Points</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Eve</td>
                <td>Jackson</td> 
                <td>94</td>
            </tr>
            <tr>
                <td>Pedro</td>
                <td>Leiva</td> 
                <td>100</td>
            </tr>
            <tr>
                <td>Jose</td>
                <td>Rivera</td> 
                <td>20</td>
            </tr>
        </tbody>
        <tfoot>
        </tfoot>
    </table>
