Select

Use 'tds-lib_form_select' class for select plugin

Option 1 Option 2 Option 3

HTML

<div class='tds-lib_form_select'>
  <span></span>
  <span><i class="tds-lib-icon-display tds-lib-16x"></i></span>
  <div>
     <span value="Option 1">Option 1</span>
     <span value="Option 2">Option 2</span>
     <span value="Option 3">Option 3</span>
  </div>
</div>

JavaScript

Get the div element with class 'tds-lib_form_select' from the DOM

var mySelect1 = new tds_lib.Select(document.getElementsByClassName("tds-lib_form_select")[0]);

Methods

Get 'select' current value

mySelect1.getSelectValue();

Get 'select' current text

mySelect1.getSelectText();

Get text associated to value

mySelect1.getTextByValue(value);

Get all option values

mySelect1.getAllValues();

Get all option texts

mySelect1.getAllTexts();

Set active option

mySelect1.setActiveOption(value);

Add select option

mySelect1.addOption(value,text);

Remove option by value

mySelect1.removeByValue(value);

Remove option by index

mySelect1.removeAt(index);

Remove all options

mySelect1.removeAll();

Trigger onChange event

mySelect1.runOnChange();

Events

onChangeOption(): Execute some action when the current select value change

mySelect1.onChangeOption(function(){
                            alert("Hello World!");
                        });
