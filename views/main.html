About tds-lib

tds-lib is a framework that's used by all projects in TEC Digital to standarize the code, and improve a better quality of the visual guidelines.



Based on bootstrap

tds-lib reuses some classes from the bootstrap framework, like the grid system or the buttons code.

Includes Accesibility features

tds-lib provides a set of accesibility features, like high contrast mode classes or aria attributes.

Full of features

tds-lib includes CSS and HTML components that allows you to reuse the code.

tds-lib is a set of user interface interactions, effects, widgets, and themes. Whether you're building highly interactive web applications or you just need to add some of the widgets below!

To add tds-lib into your website what you need is just a single line of code. You don't even have to download or install anything!

Paste the following code into the section of your site's HTML. 

<link rel='stylesheet' href='/resources/tds-lib/css/tds-lib.css'>

Paste the following code if you want to use the font icons. 

<link rel='stylesheet' href='/resources/tds-lib/css/tds-fonts.css'>

Paste the following code if you want to use the spinner, validations, tabs and javascript selects. 

<script src='/resources/tds-lib/js/tds-lib.js'></script>
