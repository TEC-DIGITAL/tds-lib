Icons

The new way of use icons in TEC-Digital it's using fonts. It gives you scalable vector icons that can instantly be customized with CSS(size, color, drop shadow, and anything that can be done CSS).

This fonts are designed to be used with inline elements (like the <span>, <div>, etc; but using a <i> is more semantically correct).


You can place Font icons just about anywhere using in the class the icon's name.

Active

If you need an icon active, just add the class tds-lib_active.

<i class="tds-lib-icon-communities tds-lib-40x tds-lib_active"></i>

No hover

If you need an icon without hover effect, just add the class tds-lib-icon-no-hover.

<i class="tds-lib-icon-communities tds-lib-40x tds-lib-icon-no-hover"></i>

Example:

Sizes

To increase icon sizes relative to their container, use the classes tds-lib-16x, tds-lib-18x, tds-lib-28x, tds-lib-32x or tds-lib-40x.

<i class="tds-lib-icon-communities tds-lib-16x"></i>
<i class="tds-lib-icon-communities tds-lib-18x"></i>
<i class="tds-lib-icon-communities tds-lib-28x"></i>
<i class="tds-lib-icon-communities tds-lib-32x"></i>
<i class="tds-lib-icon-communities tds-lib-40x"></i>

Menu icons

Menu icons with active property

Use the following structure when you want to use a menu icon with a label.

<div class="tds-lib_menu-container">
    <i id="example_menu" class="tds-lib-icon-menu1 tds-lib-icon-pointer tds-lib-32x"></i>
    <label>Menú</label>
</div>

Can also be used with HTML lists (ul and ol elements).

<ul class="tds-lib_menu-container">
    <li>
        <i id="example_menu" class="tds-lib-icon-menu1 tds-lib-icon-pointer tds-lib-32x"></i>
        <label>Menú</label>
    </li>
</ul>
