The tds-lib-spinner allows you to create input elements specialized for numbers.

How to use it?

First you have to add the webcomponent library

			
	<script src="/resources/tds-lib/lib/webcomponentsjs/webcomponents-lite.min.js"></script>

Then include the tab web component in your project. You can add this line in any part of your HTML or ADP file.

			
	<link rel="import" href="/resources/tds-lib/web_components/tds-lib-spinner/tds-lib-spinner.html">

That's all! Now you can use the tds-lib-spinner element.

Add a spinner using JavaScript

To add a spinner using JavScript you can use the traditional method

		
// Create spinner with JavaScript
var node = document.createElement("tds-lib-spinner");
// Set initial attributes
node.setAttribute("value",40);
node.setAttribute("min",40);
node.setAttribute("max",80);
document.body.appendChild(node);
Attributes
Name Attribute Description Example
value Number, Integer Value of the spinner. To change the spinner's value, change this attribute.
min Number This is the minimal value the spinner can get.
max Number This is the max value the spinner can get.
disabled If this attribute exists then the spinner will be disabled.

Spinner

Use 'tds-lib_form_spinner' class for select widget style.

HTML

<div class="input-group tds-lib_form_spinner">
    <input type="text" class="form-control" value="10">
    <div class="input-group-btn-vertical">
        <button class="tds-lib-icon-display"></button>
        <button class="tds-lib-icon-display"></button>
    </div>
</div>

JavasScript

Get the div element with class 'tds-lib_form_spinner' from the DOM

var mySpinner1 = new tds_lib.Spinner(document.getElementsByClassName("tds-lib_form_spinner")[0]);

Methods

Set the highest value

mySpinner1.setMaxValue(1000);

Set the lowest value

mySpinner1.setMinValue(-50);

Set the current value

mySpinner1.setSpinnerValue(7);

Get the current value

mySpinner1.getSpinnerValue();
